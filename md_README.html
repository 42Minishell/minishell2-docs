#minishell2

Minishell 42 project; second iteration

Compiling

Currently using CMake, biggest difference is that CMake compiles out-of-tree.

Basically this means that the build directory is not the same as the source directory.

Quick start to compiling:

# In project root
mkdir build && cd build
cmake ..
make

cmake .. only generates the makefiles, this needs to be done after editing CMakeLists.txt

Changing the source without adding new files only need a recompile using make like you're used to.
